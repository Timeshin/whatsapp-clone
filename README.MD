# Whatsapp clone

#### Application -> https://timeshin.github.io/whatsapp-clone

#### Before usage

```
npm install
```

#### Before commits

```
npm run prepare
```

### start project

#### Dev

```
npm run start:dev
```

#### Prod

```
  npm run build
  npm run start:prod
```

## Or run in docker

```
docker-compose -f docker-compose.yaml up
```
